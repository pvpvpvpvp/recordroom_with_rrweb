세션 체인 테스트

테스트 시나리오

이 테스트는 같은 세션 내에서 페이지 전환 시 레코드가 체인으로 연결되는 것을 시연합니다.
  1. 각 페이지에서 SDK가 자동으로 새 레코드를 생성합니다
  2. 이전 페이지의 레코드 ID가 previousRecordId로 전달됩니다
  3. 모든 레코드는 같은 sessionId를 공유합니다
  4. 세션 뷰에서 레코드들이 체인으로 연결된 것을 확인할 수 있습니다
